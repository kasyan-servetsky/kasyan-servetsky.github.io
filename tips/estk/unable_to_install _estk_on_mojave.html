Unable to install ESTK on Mojave (version 10.14.3)

Afer upgrading my Mac, I installed a brand new hard disk and made a clean install of Mojave. Then I successfully installed all the adobe apps I needed, but with ESTK I've got into a trouble. I downloaded it from here since it was not listed in the Adobe Creative Cloud.

When I tried to install it, I got the following error:

Clicking on the link opens this page which is of no use to me. I tried to re-download it, but got the same error.
Before ESTK did work for me on Mojave, but then, before the upgrade, I updated OS: first from El Capitan to Mojave 10.14.2 and then to 10.14.3.

Accidentally, I've found a solution:

I option + double clicked a jsx-file in the Scripts panel to open it for editing expecting it would open in a default application, or OS would suggest me to choose a program to open it with.

Instead, the Adobe Creative Cloud activated and started downloading ESTK which showed up in the list now.

After I fixed the Cannot execute script in target engine 'main'! (#1116) Can't start debug session bug, it works as usual.
