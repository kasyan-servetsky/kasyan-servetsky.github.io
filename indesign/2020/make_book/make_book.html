Make book from Excel

A quick & dirty script for InDesign 2020-2022 and Excel: two versions for Mac and Windows. Written by Kasyan on request found here.
Creates a book from the list taken from a workbook: the first column (A) in the first sheet.

Run a script and select a workbook from a location on a volume: don’t use a file on the desktop (it’s quick & dirty, as I mentioned, so currently the script can’t handle desktops).
A book with the same name as the Excel file will be created and if indd-files in the list, exist, they will be added to the book.
