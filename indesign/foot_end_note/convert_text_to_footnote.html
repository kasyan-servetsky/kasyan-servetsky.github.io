Convert text to footnote

The script was written by Jarek (Jump_Over) and modified by me (Kasyan). The source is here.

The script converts “quasi-notes” to real notes in the text.

Requirements:

  • both text and endnotes are in the same story
  • each endnote is a separate (one) paragraph
  • initially there is a white space after endnote's number
  • select either a text frame, or some text, or place the cursor into the text
