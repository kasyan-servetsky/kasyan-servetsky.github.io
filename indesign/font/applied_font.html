List font properties

Script by Roland Dreger

The script lists up to 15 properties of a font — either in style ranges, paragraphs, stories or the current insertion point in the active document. These properties are included in a new text frame on a separate layer. The respective initial position is marked with an arrowhead.

Click here to watch a short video demonstration of how it works.

Text frames / background and outline color:
The colors of the pasted text frames can be modified before program execution. This in the InDesign color palette of the active document to the desired colors 3 Position (background) and 4 Position draw (contour).

Text frame / paragraph format:
Paragraphs in the pasted text frame is the standardized format [Catch Allocated sales].
