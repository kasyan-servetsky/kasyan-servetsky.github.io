Drop cap (Lettrine)

Script for InDesign CS5
Version 1.3

Requirements:

I need a script to automatically enter a drop cap in the 'corps.chapeau' paragraph of the active story.

It uses the character style 'Drop cap'

It is 3 lines deep.

It is one character in length, except
- If the first character is preceded by quotation mark '«', then it is two characters
- If the first character is followed by an apostrophe, then it is two characters.

There is custom tracking for each letter, including apostrophe.
You can set the tracking to 0 value for all of them at the moment, I will adjust them one by one after.

In the document "5.indd", the story placed with the highest y value and lowest x value (the top left corner story) gets a drop cap 4 lines deep.

Add +100 kerning between the 2nd and 3rd character if the 2nd character is an apostrophe.
If the first character is "«" then the kerning would be placed between the 3rd (apostrophe) and 4th character.

Download the script from here.
