Insert Ending Paragraphs

Script for InDesign CS5
Version 2.1

Requirements:

A prompt asks the user to choose from "Galeries", "Événements", "Article" or "Renseignement'
The script inserts two paragraphs at the end of the story.

First paragraph style depends on what the user chooses:
Galeries = 'Gallery' paragraph style. Inserted text = "This is a gallery:"
Événements = 'Event' paragraph style. Inserted text = "This is an event:"
Article = 'Article' paragraph style. Inserted text = "More online:"
Renseignment = 'Info' paragraph style. Inserted text = "More info:"

Second paragraph style is always 'Details'.
Galeries inserted text = "www.lexpress.to/galeries"
Événements inserted text = "www.lexpress.to/calendrier"
Articles inserted text = "www.lexpress.to/####"
Renseignement inserted text = "Détails"

The paragraph styles are in the folder "Internet"

After the paragraphs are inserted and styled, the script should fit frame to content.

Full undo as well.
