Export button description settings

Script for InDesign CS6.

The script currently exports the following button settings to CSV or TXT file which can be opened in Excel:

  • Name
  • Page
  • Hidden Until Triggered
  • States
  • Show/Hide Buttons and Forms
  • Description
  • Printable

This is an ongoing project. I would like to get any feedback if someone is interested in it.
