Prime/Composite Number Highlighting

A set of scripts for InDesign by Jean-Claude Tremblay.

Prime Number Highlighting

The Prime Number Highlighting Script for Adobe InDesign enables users to easily identify and emphasize prime numbers within their documents. It scans the selected content — whether an entire document, a specific text frame, or highlighted text — using GREP patterns to locate numeric values. The script applies a distinctive character style called "Prime" (or conditional text) to all identified prime numbers.

Composite Number Highlighting

The Composite Number Highlighting Script complements the Prime Number Highlighting Script by focusing on non-prime, or composite numbers, in Adobe InDesign documents. It processes selected content, whether no selection is made, a single text frame is chosen, or a specific text is highlighted. By scanning for numeric values and applying the "Not Prime" character style (or conditional text), this script helps users easily distinguish composite numbers.

Together, these scripts provide an efficient solution for categorizing and visually managing numeric data in InDesign.
