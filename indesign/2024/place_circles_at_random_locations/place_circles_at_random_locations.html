Place circles at random locations of the page to cover text

Script for InDesign by Marc Autret

Add masking circles on the active page, that is, [Paper]-filled ovals.
diameter [opt] : diameter of each circle, either as a number in default
measurement unit, or as a val-unit string e.g. "1.5mm" or "28pt".
If not supplied, some diameter is automatically determined.
count [opt] : number of circles. If not supplied, a huge count is taken.
