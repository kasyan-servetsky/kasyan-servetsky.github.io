Update prices in the Wholesale Catalog

This script is for Windows only (but a version for Mac is also possible). Written — on request from a user at the Adobe InDesign forum — and tested in InDesign 2024 (version 19.0) by Kasyan. Based on my Get data directly from Excel function.

At the start, the script asks the user to choose an Excel file where column A corresponds to part numbers and column B to their prices. Then it reads data from Excel and updates the prices in the catalog. The prices are located in tables where the first columns contain part numbers and the last columns hold the corresponding prices.
