Select objects

This is a start-up script for InDesign. Works in CC 2018.

Author: Frans van der Geest (F vd Geest DTP). The source is here.

I just edited it a little: in particular translated it to English.

The script creates a new menu item in the Edit menu (below the Select All command) wich selects different types of objects on the spread like so:

Before

After

Note: on Windows you can hit Alt after which shortcuts (underscored letters) appear in the dialog box. Then you can hit the corresponding key to turn on/off the checkbox: for example, R for Rectangle.

You can download the script from here.
