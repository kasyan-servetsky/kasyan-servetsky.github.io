Remember Document's Position

Scripted plug-in for InDesign CS4 version 1.

A while ago, I read, in the Adobe forum, about a problem that appeared in InDesign CS4: unlike CS3, it doesn't save a document's position, which irritates some users — I assume there are quite a number of them. I thought it was quite easy to solve this with APID ToolAssistant and made a scripted plug-in which does the following:

  • When a document is saved, it saves the position of its front window.
  • When a document is opened, it restores last saved position, if such exists.

The position is saved/restored for every user independently — in case, several users work on the same file.

How to install:

  1. Download my plug-in, its totally free.
  2. Download APID ToolAssistant, the plug-in that makes all scripted plug-ins created with APID work. Detailed information about APID ToolAssistant you can read here on Rorohiko's site. It costs $25, but you can test it for 20 days as fully functional demo.
  3. Create a folder inside InDesign's Plug-Ins folder, name it whatever you like — e.g. Rorohiko — and place both downloaded plug-ins into it.
  4. Restart InDesign.

How to uninstall:

  1. Quit InDesign.
  2. Remove the folder created in the previous step.

Note: I created and tested this plug-in with InDesign CS3 on Mac and haven't tested it on CS4 because I currently don't have it in my disposal, but I hope it should work with it as well.
