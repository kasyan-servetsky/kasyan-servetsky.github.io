Resave files script

Script for InDesign CS3 version 1 & 2.

Version 1 of the script opens all InDesign documents in a selected folder one by one and resaves them into 'Resaved Files' subfolder, which is automatically created in the selected folder.
To speed up the process, InDesign doesn't show the files being opened and closed. However you can see a progress bar indicating the name of the current file, how many files have been already processed, and total number of files.
Resaved files are about twice smaller in size than original ones, thus using this script you can save a lot of disk space.

Version 2 does the same as version 1, except that it resaves original files, without creating above mentioned folder.
